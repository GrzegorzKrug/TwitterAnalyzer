# Readme

## Twitter Login
### Save your twitter token in this file secret_token.txt
### File should look like this example:

> {
>	"consumer_key" : "ABC",
>	"consumer_secret" : "ABC",
>	"access_token_key" : "ABC",
>	"access_token_secret" : "ABC"
> }
